# F1 Projects

A collection of Formula 1 projects exploring data, prediction, and simulation.

## Projects

### [chinese-gp-predictor](./chinese-gp-predictor)
Monte Carlo race predictor for the 2026 Chinese Grand Prix. Uses historical and qualifying data pulled via FastF1, a scikit-learn scoring model, and 100,000 Monte Carlo simulations to generate win probabilities. Correctly predicted the top 4 finishers in exact order (Antonelli, Russell, Hamilton, ...). Includes a live API (`/simulate`) and a Streamlit dashboard.

**Stack:** Python, FastF1, scikit-learn, NumPy, Streamlit
